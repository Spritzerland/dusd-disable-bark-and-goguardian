Removing

I currently only have instructions for Linux. If you are not on a UN*X or Linux platform, sorry!
1. Make appropriate director(ies).
user@host:~$ sudo mkdir -p /etc/opt/chrome/policies/managed
OR root@host:~# mkdir -p /etc/opt/chrome/policies/managed
2. Setting up the json file
user@host:~$ sudo cd /etc/opt/chrome/policies/managed && wget https://justin226.github.io/dusd-disable-bark-and-gogaurdian/disableschoolextensions.json
OR root@host:~# cd /etc/opt/chrome/policies/managed && wget https://justin226.github.io/dusd-disable-bark-and-gogaurdian/disableschoolextensions.json
3. Restarting Chrome
If you have chrome already running, reload policies. (Go to chrome://policies and click reload policies.)
Otherwise, start chrome.
